6. TentukanA.luas alas kerucutB.panjang garis pelukisC.luas selimut kerucutD.volume kerucutngsl:report​


Jawab:

Diketahui :

t = 20 cmd = 14 cmr = d : 2 = 14 : 2 = 7 cmπ = [tex]\frac{22}{7}[/tex]

Ditanya :

Luas alas kerucut?Panjang Garis pelukis (s)?Luas selimut kerucut?Volume kerucut?

Pembahasan :

a. Mencari luas alas kerucut

   L. alas = π x r²

               =  [tex]\frac{22}{7}[/tex] x 7²

               =  [tex]\frac{22}{7}[/tex] x 49

               = 154 cm²

  Luas alas kerucut adalah 154 cm²

b.  Menentukan panjang garis pelukis (s)

    [tex]s = \sqrt{r^2 + t^2}[/tex]

       [tex]= \sqrt{7^2 + 20^2} \\ \\= \sqrt{49 + 400} \\ \\= \sqrt{449}\\ \\ = 21,19 \ cm[/tex]

     Jadi, panjang garis pelukis adalah 21,19 cm

c. Menentukan Luas selimut kerucut

   Luas selimut = π x r x s

                         = [tex]\frac{22}{7}[/tex] x 7 x 21,19

                         = 466,18 cm²

d. Menentukan Volume kerucut (V)

   V = [tex]\frac{1}{3}[/tex] x π x r² x t

       = [tex]\frac{1}{3}[/tex] x [tex]\frac{22}{7}[/tex] x 7² x 20

       = 1.026,67 cm³
