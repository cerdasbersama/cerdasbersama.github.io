5. Terdapat sebuah kapal antar pulau yang dapat mengangkut penumpang sebanyak 240 orang. Penumpang kelas utama dapat membawa bagasi 60kg dan kelas ekonomi 20kg. Kapal tersebut hanya dapat mengangkut bagasi 7.200kg. Harga jual tiket kelas utama RP100.000,- dan tiket ekonomi Rp50.000,-. Keuntungan maksimal yang dapat diperoleh oleh pengusaha kapal tersebut adalah


Fungsi Linier.

Misalkan:

kelas utama = x

kelas ekonomi = y

Fungsi Objective:

100.000x + 50.000y

Fungsi Pembatas:

x + y ≤ 240

60x + 20y ≤ 7.200 → 3x + y ≤ 360

x ≥ 0

y ≥ 0

Penyelesaian:

1. Menentukan titik potong sumbu x dan sumbu y dan titik potong kedua fungsi

a. Fungsi x + y ≤ 240

→ titik potong sumbu x ( y = 0 )

x + 0 = 240

x = 240

titik koordinat titik potong sumbu x (240, 0)

→titik potong sumbu y ( x = 0 )

0 + y = 240

y = 240

titik koordinat titik potong sumbu x (0, 240)

b. fungsi 3x + y ≤ 360

→ titik potong sumbu x ( y = 0 )

3x + 0 = 360

x = 120

titik koordinat titik potong sumbu x (120, 0)

→titik potong sumbu y ( x = 0 )

0 + y = 360

y = 360

titik koordinat titik potong sumbu x (0, 360)

c. Kedua fungsi

x + y - 240 = 3x + y - 360

360 - 240 = 3x - x + y - y

120 = 2x

x = 60

y = 240 - 60 = 180

titik potong kedua fungs ( 60, 180)

2. Gambarkan kedua fungsi di atas, kemudian tentukan titik - titik pojoknya.

(lihat gambar pada tautan)

dari gambar dapat kita tentukan 3 titik pojok yang memenuhi batas - batas dari kedua buah fungsi tersebut yaitu:

(0,240) (60,180) dan (120,0)

3. Menentukan nilai maximum dan minimum fungsi objective

nilai maximum dan minimum dapat ditentukan dengan menguji nilai yang dihasilkan titik titik pojok tersebut terhadap fungsi objective.

a. titik (0,240)

100.000x + 50.000y

= 100.000(0) + 50.000 (240)

= 12.000.000

b. titik (60,180)

= 100.000 (60) + 50.000 (180)

= 6.000.000 + 9.000.000

= 15.000.000

c. titik (120,0)

= 100.000 (120) + 50.000 (0)

= 12.000.000

sehingga hasil maximum adalah 15.000.000 dengan komposisi 60 tiket kelas utama dan 180 tiket kelas ekonomi.

16. 1. Sebuah kapal laut mempunyai kapasitas tidak lebih dari 600 penumpang yang terdiri dari kelas eksekutif dan kelas ekonomi. Untuk kelas eksekutif hanya boleh membawa bagasi maksimal 60 kg dan kelas ekonomi maksimal 40 kg. Kapal tersebut hanya mampu membawa bagasi maksimal 30 ton. Jika harga tiket kelas eksekutif adalah Rp 700.000,-/penumpang dan kelas ekonomi Rp 500.000,-/penumpang, maka penjualan masing-masing kelas tiket yang harus terjual supaya hasil penjualan tiket sebesar-besarnya adalah Tolong ya bantu jawab


Penjelasan dengan langkah-langkah:

Ini merupakan materi dari program linear.

Berikut penjelasannya..

Kelas Eksekutif = X

Kelas Ekonomi = Y

Fungsi Kendala

X + Y ≤ 600.

60X + 40Y ≤ 30.000

Fungsi Tujuan

F(X,Y) = 700.000X + 500.000Y = ?

Ditanya X dan Y agar hasil penjualan maksimum.

Kita eliminasi dari kedua persamaan fungsi kendala.

X + Y ≤ 600

3X + 2Y ≤ 1.500

------------------------

3X + 3Y ≤ 1.800

3X + 2Y ≤ 1.500

_____________ (-)

Y ≤ 300

X ≤ 300

ada tiga titik uji untuk mencari nilai maksimum.

(terbatas dalam penjabaran gambar)

F(500,0) = 700.000 x 500 = 350.000.000

F(0,600) = 500.000 x 600 = 300.000.000

F(300,300) = 700.000(300) + 500.000(300)

= 210.000.000 + 150.000.000

= 360.000.000

jadi penjualan maksimum ada pada x = 300 dan y = 300.

Dalam kehidupan sehari-hari penerapan pembelian tiket kereta api, tiket pesawat, tiket kapal laut, pembayaran tiket tol, pembayaran listrik, pembayaran air, dan lain sebagainya. dengan operasi masukkan data ke dalam antrian, mengeluarkan data terdepan dari antrian, menghapus seluruh antrian, memeriksa apakah antrian kosong, memeriksa apakah antrian penuh. Penerapan tersebut menggunakan konsep ...​


Jawaban:

menggunakan konsep struktur

Penjelasan:

Penerapan tersebut menggunakan konsep struktur data yang disebut antrian atau queue. Antrian adalah struktur data yang bekerja dengan prinsip "first-in, first-out" (FIFO), yang berarti elemen yang pertama kali masuk ke antrian akan menjadi elemen yang pertama kali keluar dari antrian.

Dalam konteks pembelian tiket dan pembayaran, data masukkan berupa informasi yang diperlukan untuk melakukan transaksi, seperti nomor transaksi, jumlah pembayaran, atau rincian tiket. Setiap kali ada transaksi baru, data tersebut dimasukkan ke dalam antrian.

Proses mengeluarkan data terdepan dari antrian dilakukan saat transaksi diproses. Data terdepan yang keluar dari antrian adalah data yang harus diproses berikutnya. Misalnya, saat tiket kereta api atau pesawat dipesan secara online, data pemesanan masuk ke antrian, dan sistem akan mengeluarkan data tersebut secara berurutan untuk diproses oleh server.

Untuk menghapus seluruh antrian, proses ini dilakukan setelah semua transaksi selesai diproses, atau jika diperlukan penghapusan data antrian. Antrian dikosongkan dengan menghapus semua elemen yang ada di dalamnya.

Pengecekan apakah antrian kosong dilakukan untuk memastikan tidak ada lagi transaksi yang harus diproses. Jika antrian kosong, berarti tidak ada lagi data yang menunggu diproses.

Sementara itu, pengecekan apakah antrian penuh dilakukan untuk memastikan bahwa sistem tidak melebihi kapasitas antrian yang sudah ditentukan. Jika antrian penuh, transaksi baru mungkin tidak dapat diterima dan perlu dilakukan tindakan, seperti meminta pengguna untuk mencoba lagi nanti.

Dengan menggunakan konsep antrian, proses pembelian tiket, pembayaran, dan transaksi lainnya dapat dilakukan secara teratur dan sesuai urutan masuk.
