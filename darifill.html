|3× - 2| > |× + 3|

(3x - 2)^2 > (x + 3)^2

Pindahkan bilangan di ruas kanan ke ruas kiri

(3x - 2)^2 - (x + 3)^2 > 0

Faktorkan menjadi selisih kuadrat

(3x - 2 - (x + 3)) (3x - 2 + (x + 3)) > 0

(3x - 2 - x - 3) ( 3x - 2 + x + 3) > 0
