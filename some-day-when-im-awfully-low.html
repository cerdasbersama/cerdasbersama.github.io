L = 8cm

Ditanya : Luas permukaan balok tersebut ?

L=(2 x p x l) + (2 x p x t) + (2 x l x t)

=(2 x 18 x 8) + (2 x 18 x 7) + (2 x 8 x 7)

=288 + 252 + 112

= 652 cm²
