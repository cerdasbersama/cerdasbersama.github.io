8. sebuah trafo memiliki 20 lilitan primer dan 50 lilitan sekunder bila tegangan sekunder trafo 100 volt dan arus primernya 5A, berapakah tegangan primer dan arus sekendernya?​


Jawaban:

12.5 A.

Penjelasan:

Np/ Ns = Vp/ Vs = Ip/ Is

Dimana:

Np = jumlah lilitan primer

Ns = jumlah lilitan sekunder

Vp = tegangan primer

Vs = tegangan sekunder

Ip = arus primer

Is = arus sekunder

Dalam kasus ini, memiliki informasi sebagai berikut:

Np = 20 lilitan

Ns = 50 lilitan

Vs = 100 volt

Ip = 5A

dapat menggunakan persamaan untuk mencari tegangan primer (Vp):

Vp / Vs = Np / Ns

Vp / 100 = 20 / 50

Vp = (20 / 50) * 100

Vp = 40 volt

Selanjutnya, dapat menggunakan persamaan untuk mencari arus sekunder (Is):

Ip / Is = Ns / Np

5 / Is = 50 / 20

Is = (50 / 20) * 5

Is = 12.5 A

Jadi, tegangan primer trafo adalah 40 volt dan arus sekundernya adalah 12.5 A.

18. apabila taget reminder adalah 750, sedangkan jumlah actual yang di-reminder adalah 700, maka ability rate sebesar?


Jawab:

Penjelasan dengan langkah-langkah:

Ability rate adalah tingkat kemampuan seseorang atau suatu sistem untuk mencapai target. Biasanya, ability rate dihitung sebagai rasio antara jumlah yang sebenarnya dicapai dan jumlah target.

Untuk menentukan ability rate dalam hal ini, rumusnya adalah:

Ability rate = Actual / Target

Dengan target sebesar 750 dan actual sebesar 700, maka ability rate-nya adalah:

Ability rate = 700 / 750 = 0.933 atau 93,3%

Jadi, ability rate dalam kasus ini adalah 93,3%. Ini berarti bahwa seseorang atau sistem yang bertanggung jawab telah berhasil mencapai 93,3% dari target reminder.

Jadi, dengan jumlah actual reminders sebesar 700 dan target reminders sebesar 750, ability rate adalah:

(700 / 750) * 100 = 93,33%.

dik : Vp : Vs = 1 : 100
I p = 25 A
dit :
Is
jawab :
[tex] \frac{V_s}{V_p} = \frac{I_p}{I_s} [/tex]
[tex] \frac{100}{1} = \frac{25}{I_s} [/tex]
maka, Is = 25 / 100
Is = 0,25 Ampere
