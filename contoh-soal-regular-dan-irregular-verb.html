8. jelasin ttg verb irregular sm regular, dikasih contoh​


Jawaban:

Irregular verb => Kata kerja tidak beraturan dimana bentuk kata kerja kedua (past) dan bentuk kata kerja ketiga (past participle) berubah-ubah (tidak sama).

Contoh:

Wash=Washed (Artinya: Mencuci)

Want =Wanted (Artinya: Ingin)

Irregular Verbs:

Eat= Ate= Eaten (Artinya: Makan)

Go= Went= Gone (Artinya: Pergi)

See= Saw= Seen (Artinya: Melihat)

17. apa perbedaan regular verb dan irregular verb


Verb

JAWABAN:    

Perbedaan regular dan irregular verb:

Regular Verb => Kata kerja beraturan dimana bentuk kata kerja kedua (past) dan bentuk kata kerja ketiga (past participle) adalah sama.Irregular verb => Kata kerja tidak beraturan dimana bentuk kata kerja kedua (past) dan bentuk kata kerja ketiga (past participle) berubah-ubah (tidak sama).

PEMBAHASAN:  

Regular Verb  

Regular verb merupakan kata kerja berarturan. Kata kerja tersebut hanya ditambah akhiran -ed, -d, atau -ied dari bentuk dasar (invinitive). Coba perhatikan contoh berikut agar lebih paham:  

accept + ed => accepted  achieve + d => achieved  carry + => carried

Dalam menambahkan akhiran -ed terdapat beberapa aturan agar tidak terjadi kesalahan. Berikut penjelasan bagaimana cara menambahkan akhiran -ed, -d atau -ied pada verb:  

1. Verb (kata kerja) yang berakhiran huruf hidup (vowel) -e  

Berikut contohnya agar lebih paham:  

admire => admired => mengagumi  advise => advised => menasehati  

2. Verb (kata kerja) yang berakhiran huruf mati (consonant)  

Berikut contohnya agar lebih paham:  

Stop => Stopped => berhenti  Beg => Begged => memohon  Gain => Gained => memperoleh  Boil => Boiled => merebus  Listen => Listened => mendengar  Offer => Offered => menawarkan Admit => Admitted => mengakui  Control => Controlled => mengontrol  

3. Verb (kata kerja) yang berakhiran dua huruf mati (consonant)  

Untuk verb (kata kerja) yang berakhiran dua huruf mati (consonant) mendapat akhiran -ed tanpa ada perubahan pada verb (kata kerja) tersebut.  

Berikut contohnya agar lebih paham:  

Demand => Deamnded => meminta  Point => Pointed => menunjuk  

4. Verb (kata kerja) yang berakhiran huruf -y  

Berikut contohnya agar lebih paham:  

Play => Played => bemain  Enjoy => Enjoyed => menikmati  Study => Studied => belajar  Try => Tried => mencoba  

5. Verb (kata kerja) yang berakhiran huruf -ie  

Untuk verb (kata kerja) yang berakhiran huruf -ie mendapat akhiran -d di akhir verb (kata kerja) tersebut.  

Berikut contohnya agar lebih paham:  

Die => Died => meninggal  Lie => Lied => berbohong  

Irregular Verb

Irregular verb adalah kata kerja tidak berarturan. Kata kerja tersebut berubah tergantung dari penggunaannya. Untuk menguasai irregular verb maka kamu harus menghapalkannya  karena bentuknya yang berubah-ubah.

Berikut contoh irregular verb:

cast cast cast melempar, memberikan peran  catch caught caught menangkap  do did done melakukan
