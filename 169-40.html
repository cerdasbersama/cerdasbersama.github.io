14. Dari 40 siswa diambil sampel 9 orang untuk diukur tinggi badannya, di peroleh data berikut 165, 170, 169, 168, 156, 160, 175, 162, 169. Simpangan Baku sampel dari data tersebut adalah

Jadi, Simpangan Baku Sampel Dari Data Tersebut Adalah 166 Baku


Kalau Salah Maafin Yah :)

Jawab:

Simpangan baku sampel dari data tersebut adalah 5,83


Penjelasan dengan langkah-langkah:

Berikut adalah rumus simpangan baku

1. Untuk satu populasi

[tex]\sigma = \sqrt{\dfrac{\sum\limits_{i=1}^n (x_{i}-\mu)^{2}}{n}}[/tex]

2. Untuk sampel

[tex]s = \sqrt{\dfrac{\sum\limits_{i=1}^n (x_{i}-\bar{x})^{2}}{n-1}}[/tex]


Karena data yang akan diukur simpangan bakunya diambil dari sampel, maka kita akan menggunakan rumus kedua.

Pertama, cari rata-rata sampel atau [tex]\bar{x}[/tex]

[tex]\bar{x}=\dfrac{165+170+169+168+156+160+175+162+169}{9}[/tex]

[tex]\bar{x}=\dfrac{1.494}{9}[/tex]

[tex]\bar{x}=166[/tex]


Kedua, cari jumlah kuadrat selisih tiap data dengan rata-rata atau [tex]\sum\limits_{i=1}^n (x_{i}-\bar{x})^{2}[/tex]

[tex]\sum\limits_{i=1}^n (x_{i}-\bar{x})^{2}[/tex]

[tex]=(165-166)^{2}+(170-166)^{2}+(169-166)^{2}+(168-166)^{2}+(156-166)^{2}+(160-166)^{2}+(175-166)^{2}+(162-166)^{2}+(169-166)^{2}[/tex]

[tex]=(-1)^{2}+(4)^{2}+(3)^{2}+(2)^{2}+(-10)^{2}+(-6)^{2}+(9)^{2}+(-4)^{2}+(3)^{2}[/tex]

[tex]=1+16+9+4+100+36+81+16+9[/tex]

= 272


Ketiga, cari simpangan baku

[tex]s = \sqrt{\dfrac{\sum\limits_{i=1}^9 (x_{i}-\bar{x})^{2}}{9-1}}[/tex]

[tex]s = \sqrt{\dfrac{272}{9-1}}[/tex]

[tex]s = \sqrt{\dfrac{272}{8}}[/tex]

[tex]s = \sqrt{34}[/tex]

s = 5,83

Step by step explanation

_____________

Soal 1

_____________

Hasil dari (x² - 6x + 9)(x² + 3x - 40)

Jawab :

= (x² - 6x + 9)(x² + 3x - 40)

= x⁴ + 3x³ - 40x² - 6x³ - 18x² + 240x + 9x² + 27x - 360

= x⁴ + 3x³ - 6x³ - 40x² - 18x² + 9x² + 240x + 27x - 360

= x⁴ - 3x³ - 49x² + 267x - 360

_____________

Soal 2

_____________

Tentukan deret bilangan ganjil antara 134 dan 169 ?

Jawab :

Bilangan ganjil yang terletak diantara 134 dan 169 misal bilangan ganjil yang dimaksud x maka 134 < x < 169 sehingga anggota dari x sebagai berikut :

135, 137, 139, …, 167

Perhatikan bahwa bilangan-bilangan yang dimaksud membentuk barisan aritmatika dengan :

a = 135b = 2Uₙ = 167

» Uₙ = 167 maka n =…?

[tex]\begin{aligned}\rm U_n&=\rm a+(n-1)b\\167&=\rm 135+(n-1)2\\167&=\rm 135+2n-2\\167&=\rm 133+2n\\167-133&=\rm 2n\\34&=\rm 2n\\\bf n&=\bf 17\end{aligned}[/tex]

» Sehingga jumlah Bilangan-bilangan ganjil yang dimaksud :

[tex]\begin{aligned}\rm S_n&=\rm \frac{n}{2}\left(a+U_n\right)\\\rm S_{17}&=\frac{17}{2}\left(135+167\right)\\&=\frac{17}{\cancel{2}}\left(\cancel{302}\right)\\&=17\times 151\\&=\bf 2.567\end{aligned}[/tex]

∴ Diperoleh deret bilangan ganjil antara 134 dan 169 yaitu 2.567

Jawaban:

SOAL 1

(x² - 6x + 9) (x² + 3x - 40)

= x²(x² + 3x - 40) - 6x (x² + 3x - 40) + 9(x² + 3x - 40)

= x⁴ + 3x³ - 40x² - 6x³ - 18x² + 240x + 9x² + 27x - 360

= x⁴ - 3x³ - 49x² + 267x - 360

SOAL 2

a = 135

b = 2

Un = 167

Un = a + (n- 1) b

167 = 135 + (n- 1) 2

167 - 135 = 2n - 2

32 = 2n - 2

2n = 34

n = 17

Sn = n/2 (a + Un)

S17 = 17/2 (135 + 167)

S17 = 8,5 × 302

S17 = 2.567
