Suku keempat dari suatu barisan aritmatika adalah 24 dan suku kesebelasnya 64. Banyak suku barisan jika diketahui suku terakhirnya 134 adalah

Jawab:

U4 = 24

a + (4-1)b = 24

a + 3b = 24 ...(1)

U11 = 64

a + (11-1)b = 64

a + 10b = 64 ...(2)

Eliminasikan kedua persamaan:

a + 10b = 64

a + 3b = 24

_________ _

7b = 40

b = 40/7

Substitusikan b = 40/7 ke persamaan 1:

a + 3b = 24

a = 24 - 3b

a = 24 - 3 (40/7)

a = 24 - 120/7

a = 168/7 - 120/7

a = 48/7

Un = 134

a + (n-1) b = 134

48/7 + (n-1) 40/7 = 134

40/7 n + 48/7 - 40/7 = 134

40/7 n + 8/7 = 134

40n + 8 = 134 x 7

40n = 938 - 8

40n = 930

n = 930/40

n = 23,25 (pembulatan)

n = 23

U23 = 134 (suku terakhir)

Banyaknya suku barisan adalah 23
