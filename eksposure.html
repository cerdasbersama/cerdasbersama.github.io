Penjelasan:

Dalam aktivitas fotografi, mode eksposure pada kamera DSLR dan Mirrorless sangat penting untuk dipahami. Mode eksposure pada kamera merupakan salah satu opsi yang digunakan untuk memotret dan mempengaruhi kualitas pencahayaan pada hasil foto. Ada 5 mode eksposur utama, yaitu Auto, P, S, A, dan M.

Mode Otomatis (Auto) : Pilihan mode ini merupakan program yang memegang kendali penuh atas pengaturan dari inti kamera, termasuk juga pengaturan eksposur otomatis serta beberapa setting lainnya. Sangat cocok digunakan oleh fotografer pemula.

Modus P (Program) : Mode ini bekerja secara otomatis, namun fotografer masih dapat menentukan pengaturan salah satu elemen eksposur yaitu pengaturan ISO.

Mode S (Shutter Priority) : Mode ini lebih fokus pada pengaturan shutter speed yang ditentukan secara otomatis dan penyesuaiannya diatur melalui nilai shutter speed dan nilai ISO yang digunakan.

Mode A (Aperture Priority) : Mode ini lebih fokus pada pengaturan aperture yang ditentukan secara otomatis dan penyesuaiannya diatur melalui nilai aperture dan nilai ISO yang digunakan.

Mode M (Manual) : Mode ini memberikan kontrol penuh kepada fotografer untuk mengatur eksposur secara manual, termasuk pengaturan shutter speed, aperture, dan ISO.

Secara umum, mode eksposur pada kamera DSLR dan Mirrorless dibentuk berdasarkan atas 3 elemen utama yaitu Shutter Speed, Aperture, dan ISO yang saling bekerja sama. Memahami pengoperasian dari mode eksposur ini sangat penting bagi para fotografer pemula. Namun tidak semua orang dapat mengatur eksposur secara manual maka pada kamera DSLR maupun Mirrorless pun telah menyediakan mode eksposur otomatis dengan 5 mode eksposur utama.
