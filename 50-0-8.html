Ambil sebarang bilangan bulat a dan b, maka (a + b) adalah juga sebuah bilangan bulat.

Ambil sebarang bilangan bulat a dan b, maka (a – b) adalah juga sebuah bilangan bulat.

Artinya operasi pengurangan pada bilangan bulat bersifat tertutup.

contoh:

Kita pilih dua bilangan bulat, misalkan 76 dan 70, maka 76 – 70 = 6 adalah bilangan bulat.

Akan diperiksa apakah operasi perkalian pada bilangan bulat bersifat tertutup.

Ambil sebarang bilangan bulat a dan b, maka a × b = ab adalah juga sebuah bilangan bulat.

Artinya operasi perkalian pada bilangan bulat bersifat tertutup.

contoh:

Kita pilih dua bilangan bulat, misalkan 6 dan 23, maka 6 × 23 = 138 adalah bilangan bulat.

Akan diperiksa apakah operasi pembagian pada bilangan bulat bersifat tertutup.

Ambil sebarang bilangan bulat a dan b, maka:

a/b adalah suatu bilangan bulat jika terdapat suatu bilangan bulat α yang menyebabkan a = αb,

a/b adalah suatu bilangan rasional jika a ≠ αb untuk suatu bilangan bulat α.

Artinya operasi pembagian pada bilangan bulat tidak tertutup.

contoh:

Kita pilih dua bilangan bulat, misalkan 25 dan 5, maka 25 : 5 = 5 adalah bilangan bulat.

Kita pilih bilangan bulat lain, misalkan 5 dan 11, maka 5 : 11 = 5/11 adalah bilangan rasional.

Jadi, di antara operasi penjumlahan, pengurangan, perkalian, dan pembagian pada bilangan bulat, yang hasil operasinya tertutup (menghasilkan bilangan bulat juga) adalah operasi penjumlahan, pengurangan, dan perkalian.

2. Diketahui a = 51, b = 2, dan Un = 99.

Untuk mencari jumlah semua bilangan ganjil di antara 50 dan 100, pertama-tama kita cari dulu banyaknya bilangan ganjil di antara 50 dan 100, yaitu n dengan menggunakan rumus:

Un = a + (n - 1) b

99 = 51 + (n - 1)(2)

99 = 51 + 2n - 2

99 = 49 + 2n

2n = 99 - 49

n = 25.

Selanjutnya dengan rumus jumlah n suku pertama suatu barisan aritmatika,

Sn = ½n(2a + (n - 1)b)

Sn = 25/ 2n(2,51 + (25 - 1)2)

Sn = 25(51 + 24) = 25,75 = 1.875

Jadi, Jumlah semua bilangan ganjil antara 50 dan 100 adalah 1.875.

PemfaktoranMelengkapkan kuadratMenggunakan rumus ABC atau Kuadratis

3x + 2 = 0

3x = - 2

x = - ⅔

Untuk akar-akar kedua

x - 4 = 0

x = 4

HP = { x₁, x₂ }

HP = { - ⅔, 4 }
