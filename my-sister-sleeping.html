4. My mom ... (sleep) when I got home. A.slept.B.is sleeping.C.was sleeping.D.were sleeping.​


C. Was sleeping

↪My mom (was sleeping) when i got home.

PEMBAHASAN:

Past Continuous Tense merupakan jenis kalimat yang membicarakan tentang sesuatu yang sedang terjadi sebelum dan atau setelah waktu tertentu pada masa lalu/lampau.

➡Rumus:

(+) S + Were/was + V -ing + O

(-) S + Were/was + not + V -ing + O

(?) Was/were + S + V -ing + O + ?

Penjelasan:

Terjemahan: Ibuku (sedang tertidur) saat aku datang kerumah.

- My mom -> Subjek tunggal (Was)- Kalimat ini sudah jelas kalimat dari Past continuous tense karena terdapat kata "when" (saat) berarti sedang membicarakan tentang aktifitas atau situasi yang sedang terjadi pada masa lampau.

Pembahasan

Past continuous tense atau past progressive tense adalah sebuah kata kerja yang digunakan untuk menyatakan kejadian atau peristiwa yang secara jelas dilakukan di masa lampau.

Rumus past continuous(+) S + to be (was/were) + V-ing + O(-) S + to be (was/were) + not + V-ing + O(?) To be (was/were) + S + V-ing + O

Penggunaan was dan were

Was, digunakan untuk subjek I, she, he, itWere, digunakan untuk subjek you, we, they

PEMBAHASAN"Present Continuous Tense"

Rumus

(+) Subject + To be + (V1 + ing) + Object

(-) Subject + Don't / Doesn't + To be + (V1 + ing) + Object

(?) To be + Subject + (V1 + ing) + Object?

Keterangan

Untuk Subject --> I, to be nya "am"Untuk Subject --> You, They We, to be nya "are"Untuk Subject --> She, He, It, to be nya "is"

Untuk kata "My baby" termasuk kategori she/he, sehingga menggunakan to be "is"
