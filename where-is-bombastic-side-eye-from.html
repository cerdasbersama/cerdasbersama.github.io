PEMBAHASANPLURAL NOUNS

Kata benda yang jumlahnya lebih dari satu. Biasanya untuk menunjukan bahwa benda tersebut jumlahnya lebih dari satu (banyak), maka diakhir kata bendanya terdapat tambahan huruf -s/es/ies/ves. Plural nouns terbagi menjadi 2, yaitu :

Regular nounsIrregular nounsRegular Nouns

Kata benda beraturan yang perubaan bentuk ke pluralnya hanya menambahkan s/es/ies/ves. Lantas, bagaimana ciri-ciri kata benda yang bisa ditambahkan s/es/ies/ves? Berikut ini adalah penjelasannya :

Tambahkan -es

Jika kata bendanya berakhiran huruf o, s, ch, sh atau x.

Contoh :

Tomato => TomatoesBus => BusesBench => BenchesBrush => BrushesFox => Foxes

Tambahkan -ies

Jika kata bendanya berakhiran huruf -y, tapi sebelum huruf -y ada huruf mati. Huruf -y kemudian dihilangkan.

Contoh :

City => CitiesLibrary => Libraries

Tambahkan -ves

Jika kata bendanya berakhiran huruf -f atau-fe. Maka -f atau -fe dihilangkan.

Contoh :

Calf => CalvesWife => Wives

Tambahkan -s

Jika kata bendanya tidak memiliki huruf akhir seperti yang dijelaskan di atas.

Contoh :

Key => KeysTeacher => Teachers

Irregular Nouns

Kata benda tidak beraturan yang perubahan bentuk pluralnya berbeda dari bentuk singularnya.

Contoh :

Foot => FeetMan => MenMouse => MiceChild => Children

KESIMPULAN

Berikut ini adalah kata benda dari soal diatas yang termasuk regular dan irregular nouns.

REGULAR NOUNS

Tambahan -es

Dish => DishesBox => BoxesSandwich => SandwichesMatch => MatchesAvocado => AvocadoesBeach => BeachesLunch => LunchesWatch => Watches

Tambahan -ies

Secretary => SecretariesFamily => FamiliesDictionary => DictionariesCity => Cities

Tambahan -s

Eye => EyesDay => DaysBag => BagsFlower => FlowersVegetable => VegetablesSchool => SchoolsHotel => Hotels

IRREGULAR NOUNS

Child => Children
