37. tolong jawa, hari ini dikumpul


Diketahui akar-akar persamaan kuadrat 2x² - x - 3 = 0 adalah α dan β. Tentukan PKB yang akar-akarnya α + β dan α.β

Jawab :

x3 = α+β = -(-1/2) = 1/2

x4 = α.β = -3/2

PKB : x² - (1/2 + (-3/2))x + (1/2 . (-3/2)) =0

x² - (-2/2) + (-3/4) = 0

x² + 1 - 3/4 = 0

4x² + 4 - 3 = 0
