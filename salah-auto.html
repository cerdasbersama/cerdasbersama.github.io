27. Bantuin matematika,yang salah auto report​


PEMBAHASAN

SOAL 1

20, 17, 14, 11, ...

a = 20

b = 17 - 20 = -3

Menentukan rumus suku ke-n

[tex]un = a + (n - 1).b[/tex]

[tex]un = 20 + (n - 1). - 3[/tex]

[tex]un = 20 - 3n + 3[/tex]

[tex]un = (20 + 3) - 3n[/tex]

[tex]un = 23 - 3n[/tex]

Kesimpulan:

Jadi, rumus suku ke-n dari barisan tersebut adalah 23 - 3n (opsi A).

SOAL 2

Menentukan GSPD

[tex]GSPD = \sqrt{ {20}^{2} - {(12 + 4)}^{2} } [/tex]

[tex]GSPD = \sqrt{400 - {16}^{2} } [/tex]

[tex]GSPD = \sqrt{400 - 256} [/tex]

[tex]GSPD = \sqrt{144} [/tex]

[tex]GSPD = 12 \: cm[/tex]

Kesimpulan:

Jadi, panjang garis singgung persekutuan dalam adalah 12 cm (opsi C).

32. Ndang sat set maszeee....salah auto hpuss!​


Jawab:

(2, –25)

PEMBAHASAN

Titik Puncak Fungsi Kuadrat

Misalkan (p, q) adalah koordinat titik puncak sebuah fungsi kuadrat f(x) = ax² + bx + c.

Maka:

p = –b/(2a)  ← ingat cara menentukan sumbu simetriq = –D/(4a) = –(b² – 4ac)/(4a)

Diberikan fungsi f(x) = x² – 4x – 21

⇒ a = 1, b = –4, c = –21

p = –(–4)/(2·1) = 4/2 = 2

q = –[(–4)² – 4·1·(–21)]/(4·1)

   = –(16 + 84)/4

   = –(4 + 21)

   = –25

Nilai q juga bisa ditentukan dengan menghitung f(p).

f(2) = 2² – 4(2) – 21 = 4 – 8 – 21 = –25

Sehingga diperoleh:

⇒  Koordinat titik puncak = (p, q) = (2, –25)

KESIMPULAN

∴  Koordinat titik puncak dari f(x) = x² – 4x – 21 adalah (2, –25).
