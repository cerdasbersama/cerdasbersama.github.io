Alasan:

- Soal ini meminta kita untuk memberikan input adonan ke lubang input yang sesuai berdasarkan petunjuk dari kata-kata MAURITANIA MALI NIGER CHAD SEMEN.

- Jika kita ambil huruf pertama dari setiap kata tersebut akan membentuk kata MALIN, yang merupakan kode untuk lubang input nomor 2.

- Kata-kata tersebut juga mengarahkan kita untuk memberikan input berbentuk lingkaran (), bukan segitiga.

Cara saya menyelesaikan masalah:

1. Baca petunjuk kata-kata dengan teliti

2. Ambil huruf pertama dari setiap kata sehingga membentuk kata baru MALIN

3. Kata MALIN merupakan kode untuk lubang input nomor 2

4. Petunjuk kata-kata juga mengarahkan bentuk input lingkaran

5. Jadi jawabannya adalah memberikan input lingkaran ke lubang nomor 2
