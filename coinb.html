3. Helen and Ivan had the same number of coins. Helen had a number of 50-cent coins and 64 20-cent coins. These coins had a mass of 1.134kg. Ivan had a number of 50-cents coins and 104 20-cent coins. A) given that each 50-cent coin is 2.7g more heavier than a 20-cent coin, what is the mass of Ivan's coin in kg?​


Jawab:

1.026 kg

Penjelasan dengan langkah-langkah:

Let n be the number of Helen's 50-cent coins.

It is given that:

Helen and Ivan had the same number of coinsHelen had a number of 50-cent coins and 64 20-cent coinsIvan had a number of 50-cents coins and 104 20-cent coins

104 – 64 = 40

From these facts, we know that Ivan had 40 20-cent coins more than Helen's. Therefore, we can get that the number of Ivan's 50-cent coins which equals to:

n – 40

Let x be the mass of each 50-cent coin, and y be the mass of each 20-cent coin.

At Helen's side, we can get that

nx + 64y = 1.134kg = (1.134 * 1000) g

⇔  nx + 64y = 1,134     .....(i) (in grams)

Given that each 50-cent coin is 2.7g heavier than a 20-cent coin, we can get that

x = y + 2.7      (in grams)

Substitute x to equation (i):

n(y + 2.7) + 64y = 1,134     .....(ii)

Let m be the mass of Ivan's coins, we can get that:

(n – 40)x + 104y = m

⇔  nx – 40x + 104y = m

Then we substitute x with y + 2.7.

⇔  n(y + 2.7) – 40(y + 2.7) + 104y = m

⇔  n(y + 2.7) – 40y – 40(2.7) + 104y = m

⇔  n(y + 2.7) + 64y – 108 = m

Substitute n(y + 2.7) + 64y with 1,134 we got in equation (ii):

⇔  1,134 – 108 = m

⇔  m = 1,026 g = (1,026 / 1,000) kg

⇔ m = 1.026 kg

CONCLUSION:

∴  The mass of Ivan's coins is 1.026 kg.

15. Nick, Linda and Cindy have 16 780 coins altogether. Linda has 2860 fewer coins than Nick. Cindy has three times as many coins as Nick. How many coins does Cindy have?​


Let's represent the number of coins each person has with variables:

Let N be the number of coins Nick has.

L be the number of coins Linda has.

C be the number of coins Cindy has.

We are given three pieces of information:

1. Nick, Linda, and Cindy have 16,780 coins altogether, which can be expressed as:

N + L + C = 16,780

2. Linda has 2,860 fewer coins than Nick, which can be expressed as:

L = N - 2,860

3. Cindy has three times as many coins as Nick, which can be expressed as:

C = 3N

Now, we can set up a system of equations using these relationships:

1. N + L + C = 16,780

2. L = N - 2,860

3. C = 3N

We can use substitution to solve this system. First, we'll substitute L and C from equations 2 and 3 into equation 1:

N + (N - 2,860) + 3N = 16,780

Now, combine like terms:

5N - 2,860 = 16,780

Add 2,860 to both sides:

5N = 16,780 + 2,860

5N = 19,640

Now, divide by 5:

N = 19,640 / 5

N = 3,928

So, Nick has 3,928 coins. Now, we can find how many coins Cindy has:

C = 3N

C = 3 * 3,928

C = 11,784

Cindy has 11,784 coins.
