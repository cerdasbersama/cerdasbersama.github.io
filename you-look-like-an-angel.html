Jawaban:

d. How does he look like?

PEMBAHASAN Present Tense

Digunakan untuk menyatakan Kegiatan yang dilakukan secara berulang, rutin, kebiasaan dan fakta.

Rumus Present Tense

(+) s + v1 (-s/es) + complement

(-) S + do/does + not + complement

(?) WH + do/DOES + S + V1 + Complement?

KESIMPULAN

Look – he – how – like – ? - does

=> How does he look like? (D) ✔️

Artinya: Bagaimana penampilan dia?

Keterangan:WH question => howDo/does => doesSubject => heverb-1 => lookComplement => likePelajari Lebih Lanjut
