Seorang pedaganh pakaian ingin membeli pakaian pria dan pakaian wanita tidak lebih dari 60 stel .harga satu stel pakaian pria 100000 dan harga satu stel pakaian wanita 1500000 .jika uang yang dimiliki pedagang tersebut hanya 7750000 banyak pakaian pria dimisalakan x stel dan banyak pakaian wanita y stel ,maka model matematika persoalan di atas adalah.


Soal ini bisa dikerjakan menggunakan sistem persamaan linear dua variabel baik dengan menggunaakan metode eliminasi maupun metode substitusi. Sistem persamaan linier dua variabel adalah persamaan yg memiliki dua variabel yg saling berkaitan antarpersamaan sehingga sistem persamaan tersebut akan memiliki pernyelesaian yg tepat.

Pembahasan

Dimisalkan :

Banyak pakaian pria = x stel

Banyak pakaian wanita = y stel

Seorang pedagang pakaian ingin membeli pakaian pria dan pakaian wanita tidak lebih dari 60 stel : x + y = 60

Harga satu stel pakaian pria Rp 100.000 dan harga satu stel pakaian wanita Rp 150.000 dengan uang yang dimiliki pedagang tersebut hanya Rp 7.750.000, maka :

100.000x + 150.000y = 7.750.000

-> masing - masing ruas bagi dengan 10.000

( 100.000 : 10.000 )x + ( 150.000 : 10.000 )y = 7.750.000 : 10.000

10x + 15y = 775  

-> bagilah masing - masing dengan 5  

( 10 : 5 )x + ( 15 : 5 )y = 775  : 5

2x + 3y = 155

Didapatkan dua persamaan yaitu :

x + y = 60 ........... ( 1 )

2x + 3y = 155 ........... ( 2 )

Eliminasikan persamaan (2) dan persamaan (1)

2x + 3y = 155 | x 1 | 2x + 3y = 155  

 x +    y = 60   | x 2 | 2x + 2y = 120

------------------------------------- -

2x + 3y = 155  

2x + 2y = 120

----------------- -  

3y - 2y = 155 - 120

y = 35

Substitusikan y = 35 ke persamaan (1)

x + y = 60  

x + 35 = 60

x = 60 - 35

x = 25

Maka diperoleh bahwa banyak pakaian pria = 25 stel dan banyak pakaian wanita = 35 stel.

Model matematika untuk persoalan di atas adalah : x + y = 60 dan 2x + 3y = 155.  

Noted : Satu stel pakaian wanita Rp 150.000 karena jika satu stel pakaian wanita Rp 1.500.000 maka tidak didapatkan himpunan penyelesaian untuk persamaan tersebut.

Sebuah pabrik garmen memproduksi pakaian pria dan wanita. Hasil penjualan 100 pakaian pria dan 140 pakaian wanita sebesar Rp 8.200.000, sedangkan hasil penjualan 150 pakaian pria dan 80 pakaian wanita sebesar Rp 8.400.000. Jika x menyatakan banyak pakaian pria dan y menyatakan banyak pakaian wanita, model matematika yang sesuai dengan permasalahan tersebut adalah:

5x + 7y = 410.000

15x + 8y = 840.000

x , y > 0

Penjelasan dengan langkah-langkah:

Langkah untuk mengubah soal cerita menjadi model matematika adalah dengan menuliskan semua hal penting dari soal, menentukan variabelnya, membuat tabel dan membuat model matematikanya. Setelah mengetahui langkah dasar untuk membuat model matematika kita bisa menyelesaikan kedua soal diatas. Tabel dari pernyataan di atas dapat dilihat pada lampiran.

Pada soal di atas dikatakan bahwa hasil penjualan 100 pakaian pria dan 140 pakaian wanita sebesar Rp 8.200.000, maka persamaannya adalah:

100x + 140y = 8.200.000 disederhanakan menjadi 5x + 7y = 410.000

Pada pernyataan kedua dikatakan bahwa hasil penjualan 150 pakaian pria dan 80 pakaian wanita sebesar Rp 8.400.000, maka persamaannya adalah:

150x + 80y = 8.400.000 disederhanakan menjadi 15x + 8y = 840.000

Harga dari pakaian pria maupun wanita pasti lebih besar dari 0, maka:

x , y > 0

Jadi model matematika yang tepat untuk soal di atas adalah:

5x + 7y = 410.000

15x + 8y = 840.000

x , y > 0

31. Seorang pedaganh pakaian ingin membeli pakaian pria dan pakaian wanita tidak lebih dari 60 stel .harga satu stel pakaian pria 100000 dan harga satu stel pakaian wanita 1500000 .jika uang yang dimiliki pedagang tersebut hanya 7750000 banyak pakaian pria dimisalakan x stel dan banyak pakaian wanita y stel ,maka model matematika persoalan di atas adalah.


Soal ini bisa dikerjakan menggunakan sistem persamaan linear dua variabel baik dengan menggunaakan metode eliminasi maupun metode substitusi. Sistem persamaan linier dua variabel adalah persamaan yg memiliki dua variabel yg saling berkaitan antarpersamaan sehingga sistem persamaan tersebut akan memiliki pernyelesaian yg tepat.

Pembahasan

Dimisalkan :

Banyak pakaian pria = x stel

Banyak pakaian wanita = y stel

Seorang pedagang pakaian ingin membeli pakaian pria dan pakaian wanita tidak lebih dari 60 stel : x + y = 60

Harga satu stel pakaian pria Rp 100.000 dan harga satu stel pakaian wanita Rp 150.000 dengan uang yang dimiliki pedagang tersebut hanya Rp 7.750.000, maka :

100.000x + 150.000y = 7.750.000

-> masing - masing ruas bagi dengan 10.000

( 100.000 : 10.000 )x + ( 150.000 : 10.000 )y = 7.750.000 : 10.000

10x + 15y = 775  

-> bagilah masing - masing dengan 5  

( 10 : 5 )x + ( 15 : 5 )y = 775  : 5

2x + 3y = 155

Didapatkan dua persamaan yaitu :

x + y = 60 ........... ( 1 )

2x + 3y = 155 ........... ( 2 )

Eliminasikan persamaan (2) dan persamaan (1)

2x + 3y = 155 | x 1 | 2x + 3y = 155  

 x +    y = 60   | x 2 | 2x + 2y = 120

------------------------------------- -

2x + 3y = 155  

2x + 2y = 120

----------------- -  

3y - 2y = 155 - 120

y = 35

Substitusikan y = 35 ke persamaan (1)

x + y = 60  

x + 35 = 60

x = 60 - 35

x = 25

Maka diperoleh bahwa banyak pakaian pria = 25 stel dan banyak pakaian wanita = 35 stel.

Model matematika untuk persoalan di atas adalah : x + y = 60 dan 2x + 3y = 155.  

Noted : Satu stel pakaian wanita Rp 150.000 karena jika satu stel pakaian wanita Rp 1.500.000 maka tidak didapatkan himpunan penyelesaian untuk persamaan tersebut.
