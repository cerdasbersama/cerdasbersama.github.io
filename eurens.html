6. Permutasi kata "Eurena" adalah ......Yang like aku sebanyak banyaknya akan di tanyakan permutasinya.​


Permutasi kata "Eurena" adalah 360 susunan

_________________

Rumus:

Permutasi (jika ada unsur ganda) => P = N!÷K!

Permutasi (jika tanpa unsur ganda) => P = N!

•

Keterangan:

N = Jumlah Unsur

K = Unsur Ganda

P = Permutasi

PENYELESAIAN:

EURENA

N = 6

K = 2 (E)

P = N!/K!

P = 6!/2!

P = 6×5×4×3×2×1/2×1

P = 720/2

P = 360

34. Jelaskan pengertian dari stephalococcus aureus​


Jawaban:

Staphylococcus aureus merupakan bakteri Gram-Positif berbentuk bulat berdiameter 0,7-1,2 μm, tersusun dalam kelompok-kelompok yang tidak teratur seperti buah anggur, fakultatif anaerob, tidak membentuk spora, dan tidak bergerak.

Staphylococcus aureus adalah bakteri yang dapat menyebabkan berbagai jenis infeksi di manusia, termasuk infeksi kulit, infeksi saluran pernapasan, dan sepsis. Bakteri ini dapat bertahan hidup di luar tubuh dan dapat menyebar melalui kontak dengan permukaan yang terkontaminasi atau melalui kontak pribadi dengan seseorang yang terinfeksi. Beberapa strain dari S. aureus juga dapat menyebabkan infeksi yang resisten terhadap antibiotik, yang dapat menyulitkan pengobatan.

36. fungsi dari bacteri staphylococcus aureus


Bakteri Staphylococus Aureus bisa didapati pada permukaan kulit yang berfungsi sebagai flora normal, terutama pada sekitar mulut, hidung, sekitar anus dan alat kelamin. 40. berilah 2 contoh file ektens dan jelaskan​


Jawaban:

Ekstensi file adalah tanda yang membedakan antara satu jenis file dengan jenis file lainnya. Misalnya, file gambar akan memiliki ekstensi jpg, gif, png, dan lain-lain. Sedangkan untuk file video akan memiliki ekstensia mpeg, avi, mp4, wmv, dan lain-lain. File di dalam komputer dengan ekstensi yang berbeda-beda.

Jawaban:

Berikut adalah dua contoh file dengan ekstensi yang berbeda:

1.File dengan ekstensi .docx: Ini adalah format file yang digunakan oleh Microsoft Word untuk menyimpan dokumen teks. File .docx adalah format dokumen yang lebih baru dan lebih canggih dibandingkan dengan format .doc yang lebih lama. File .docx dapat berisi teks, gambar, tabel, grafik, dan berbagai elemen lainnya. Format ini mendukung fitur-fitur seperti pemformatan teks, pengaturan paragraf, penomoran halaman, dan lain-lain. File .docx dapat dibuka dan diedit menggunakan program seperti Microsoft Word, LibreOffice, atau Google Docs.

2.File dengan ekstensi .jpg: Ini adalah format file gambar yang paling umum digunakan di web dan media digital. File .jpg menggunakan metode kompresi yang disebut “Joint Photographic Experts Group” (JPEG) untuk mengurangi ukuran file tanpa mengorbankan kualitas gambar yang terlalu banyak. Format ini cocok untuk menyimpan foto dan gambar dengan banyak warna dan detail. File .jpg dapat dibuka dan dilihat menggunakan program tampilan gambar standar seperti Windows Photo Viewer, Adobe Photoshop, atau aplikasi pengolah gambar lainnya.
