Sistem pernapasan sendiri diatur oleh pusat pernapasan yang bertanggung jawab untuk mengontrol pernapasan secara otomatis sehingga kita tidak akan lupa bernapas baik saat beraktivitas ataupun saat tidur. diketahui
BK' = 3
AK' = √AB² + BK'²
       = √4² + 3²
       = √16 + 9
       = √25
       = 5

KK' = 1 cm
AK = √AK'² + KK'²
       = √5² + 1²
       = √25 + 1
       = √26

LF = √LE²²+ EF²
LF = √3² + 4²
LF = √9 + 16
      = √25
     = 5

FG = 1 cm

LG = √LF² + FG²
       = √5² + 1²
       = √25 + 1
       = √26
