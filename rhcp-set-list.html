15. apa perbedaan single linked list dengan double linked list serta contoh masing masing


jawaban

Perbedaan Singly Linked List, Double Linked List, Dan Circular Linked List

1. Singly Linked List merupakan suatu linked list yang hanya memiliki satu variabel pointer saja. Dimana pointer tersebut menunjuk ke node selanjutnya, biasanya field pada tail menunjuk ke NULL.

2. Doubly Linked List merupakan suatu linked list yang memiliki dua variabel pointer yaitu pointer yang menunjuk ke node selanjutnya dan pointer yang menunjuk ke node sebelumnya. Setiap head dan tailnya juga menunjuk ke NULL.

3. Circular Linked List merupakan suatu linked list dimana tail (node terakhir) menunjuk ke head (node pertama). Jadi tidak ada pointer yang menunjuk

24. Jumlah set dalam permainan bulu tangkis adalah.... a. 1 dan 3 set; b. 2 dan 3 set; c. 3 dan 4 set; d. 4 dan 6 set; e. 3 dan 5 set.


Dalam permainan bulu tangkis, setiap pertandingan terdiri dari (B) 2 dan 3 set. Set adalah jumlah babak dalam satu pertandingan bulu tangkis.

Pembahasan

Permainan bulu tangkis adalah salah satu olahraga bola besar yang dapat dimainkan secara tunggal atau beregu. Tujuan dari permainan bulu tangkis adalah untuk mematikan bola di area permainan lawan dan memenangkan pertandingan. Permainan bulu tangkis dimainkan di lapangan tertutup yang terhindar dari hembusan angin.

Dalam permainan bulu tangkis, setiap pertandingan dimainkan dalam 2 atau 3 babak. Jika pemain atau regu berhasil memenangkan 2 set awal, maka dinyatakan sebagai pemenang dan pertandingan berhasil. Namun jika kedua pemain atau regu imbang di 2 set awal, maka pertandingan dilanjutkan pada set ke 3 sebagai set penentuan.

Dalam permainan bulu tangkis, terdapat beberapa teknik dasar yang harus dikuasai oleh para pemain, yakni.

Teknik memegang raket Teknik posisi badan Teknik langkah kaki Teknik pukulan Teknik servis

<ul>

<ol>

<dl>

 <dt>Coffee</dt>

 <dd>- black hot drink</dd>

 <dt>Milk</dt>

 <dd>- white cold drink</dd>

</dl>

34. Buatlah fungsi pada python yang dapat memproses list integer kemudian mengembalikan integer-integer unik dari list tersebut tanpa menggunakan fungsi bawaan seperti “set()” dan “unique()”. Gunakanlah conditional-if dan loops!


Jawaban:

Berikut adalah contoh implementasi fungsi pada Python yang dapat memproses list integer dan mengembalikan integer-integer unik dari list tersebut tanpa menggunakan fungsi bawaan seperti "set()" atau "unique()":

```python

def get_unique_integers(numbers):

unique_numbers = []

for num in numbers:

if num not in unique_numbers:

unique_numbers.append(num)

return unique_numbers

# Contoh penggunaan fungsi

numbers = [1, 2, 2, 3, 4, 4, 5, 6, 6, 7]

unique_numbers = get_unique_integers(numbers)

print(unique_numbers)

```

Output:

```

[1, 2, 3, 4, 5, 6, 7]

```

Pada contoh di atas, kita mendefinisikan fungsi `get_unique_integers` yang menerima parameter `numbers` yang merupakan list integer. Fungsi tersebut menggunakan loop `for` untuk iterasi melalui setiap elemen dalam list `numbers`. Kemudian, menggunakan conditional `if` untuk memeriksa apakah elemen tersebut sudah ada dalam list `unique_numbers`. Jika belum ada, maka elemen tersebut ditambahkan ke list `unique_numbers`. Akhirnya, fungsi mengembalikan list `unique_numbers` yang berisi integer-integer unik dari list input.
