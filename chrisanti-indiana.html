16. Indiana menabung di bank Kak sebesar 400.000 pada saat ini ia menabung suku bunga di bank k sebesar 3%. Berapakah jumlah uang Andi anak jika ia menabung selama : a) 15 bulan b) 3 tahun ​


Jawaban:

Untuk menghitung jumlah uang yang akan dimiliki oleh Indiana setelah menabung selama periode tertentu dengan suku bunga 3%, kita dapat menggunakan rumus bunga sederhana:

Jumlah Akhir = Jumlah Awal + (Jumlah Awal x Suku Bunga x Waktu)

A. Untuk menabung selama 15 bulan (1 tahun 3 bulan):

Jumlah Awal (P) = Rp 400.000

Suku Bunga (r) = 3% atau 0,03 (dalam bentuk desimal)

Waktu (t) = 15/12 tahun (karena 15 bulan setara dengan 1,25 tahun)

Jumlah Akhir = 400.000 + (400.000 x 0,03 x 1,25)

Jumlah Akhir = 400.000 + 15.000

Jumlah Akhir = Rp 415.000

Jadi, jumlah uang yang akan dimiliki Indiana setelah menabung selama 15 bulan adalah Rp 415.000.

B. Untuk menabung selama 3 tahun:

Jumlah Awal (P) = Rp 400.000

Suku Bunga (r) = 3% atau 0,03 (dalam bentuk desimal)

Waktu (t) = 3 tahun

Jumlah Akhir = 400.000 + (400.000 x 0,03 x 3)

Jumlah Akhir = 400.000 + 36.000

Jumlah Akhir = Rp 436.000

Jadi, jumlah uang yang akan dimiliki Indiana setelah menabung selama 3 tahun adalah Rp 436.000.

34. 1, Indiana was listening music at 7 o'clock last night2, the police caught the thievesjadikan ke passive form kak​


ྀ༘ Passive and Active Voice ྀ༘

━━━━━━━━━━━━━━━━━━━━━━━━

❶ Music was being listened by Indiana at 7 o'clock last night.

❷ The thieves were caught by the police.

◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌

PEMBAHASANNomor ❶

Active Voice: Indiana was listening music at 7 o'clock last night.

➧ Passive Voice: Music was being listened by Indiana at 7 o'clock last night.

Pada nomor ❶ bentuk kalimat merujuk pada bentuk waktu Simple Past Continuous Tense yang memiliki pola kalimat sebagai berikut:

⚜ Active Voice

(+) S + was/were + V-ing + O + C(–) S + was/were + not + V-ing + O + C(?) Was/were + S + V-ing + O + C

⚜ Passive Voice

(+) S + was/were + being + V3 + by agent + C(–) S + was/were + not + being + V3 + by agent + C(?) Was/were + S + being + V3 + by agent + C + ?

Nomor ❷

Active Voice: The police caught the thieves.

➧ Passive Voice: The thieves were caught by the police.

Pada nomor ❷ bentuk kalimat merujuk pada bentuk waktu Simple Past Tense yang memiliki pola kalimat sebagai berikut:

⚜ Active Voice:

(+) S + V2 + O + C(–) S + did + not + V1 + O + C(?) Did + S + V1 + O + C ?

⚜ Passive Voice:

(+) S + was/were + V3 + by agent + C(–) S + was/were + V3 + by agent + C(?) Was/were + S + V3 + by agent + C + ?

— Keterangan: —

"Was" digunakan untuk subjek: I, he, she, it dan nama makhluk/benda banyak (Anh and Dys, the cats, or the plates)."Were" digunakan untuk subjek: You, we, they dan nama makhluk/benda banyak (Anh and Dys, the cats, or the plates).

Subject : English

Category : Reported Speech

Level : JHS VIII

Code : 8.5

------

Yang perlu diingat saat membuat kalimat langsung menjadi kalimat tidak langsung :

1. Jika reporting verb dalam past tense, tense-nya mundur. Present jadi Past, Past jadi Past Perfect.

2. I adalah yang bicara, YOU adalah yang diajak bicara.

3. Perubahan keterangan waktu.

4. Kalimat positif dimulai dengan (THAT). Kalimat tanya dimulai dengan IF/WHETHER (untuk yes-no question) dan KATA TANYA. Kalimat perintah dimulai dengan TO + infinitive.

5. This → that, these → those, here → there

--------------

2. Wawan asked Indiana how long she had stayed in that hotel.
