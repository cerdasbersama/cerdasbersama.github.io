10. ppppppppppppppppppppppppppppersamaan trigonometri fungsi cos​


Jawab:

HP : { [tex]\frac{1}{2}[/tex]π, [tex]\frac{1}{6}[/tex]π }

Penjelasan dengan langkah-langkah:

2 cos (x - [tex]\frac{1}{3}[/tex]π) = [tex]\sqrt{3}[/tex]

______________ : 2 (kedua ruas dibagi 2)

cos (x - [tex]\frac{1}{3}[/tex]π) = [tex]\frac{\sqrt{3}}{2}[/tex]

Ingat bahwa cos [tex]\frac{1}{6}[/tex]π = [tex]\frac{\sqrt{3}}{2}[/tex] , maka:

cos (x - [tex]\frac{1}{3}[/tex]π) = cos [tex]\frac{1}{6}[/tex]π

Persamaan trigonometri untuk fungsi cos x = cos α adalah:

1. x = α + 2πk

2. x = -α + 2πk

Maka persamaan trigonometri untuk fungsi cos (x - [tex]\frac{1}{3}[/tex]π) = cos [tex]\frac{1}{6}[/tex]π adalah :

1. x - [tex]\frac{1}{3}[/tex]π = [tex]\frac{1}{6}[/tex]π + 2πk

  x = [tex]\frac{3}{6}[/tex]π + 2πk

  x = [tex]\frac{1}{2}[/tex]π + 2πk

  Untuk nilai k = -1, maka :

      x = [tex]\frac{1}{2}[/tex]π + 2π(-1)

      x = [tex]\frac{1}{2}[/tex]π - 2π

      x = [tex]\frac{1}{2}[/tex]π - [tex]\frac{4}{2}[/tex]π

      x = [tex]-\frac{3}{2}[/tex]π (Tidak Memenuhi karena nilai [tex]-\frac{3}{2}[/tex]π di luar interval 0 ≤ x ≤ 2π)

  Untuk nilai k = 0, maka :

      x = [tex]\frac{1}{2}[/tex]π + 2π(0)

      x = [tex]\frac{1}{2}[/tex]π + 0

      x = [tex]\frac{1}{2}[/tex]π (Memenuhi karena nilai [tex]\frac{1}{2}[/tex]π di dalam interval 0 ≤ x ≤ 2π)

  Untuk nilai k = 1, maka :

      x = [tex]\frac{1}{2}[/tex]π + 2π(1)

      x = [tex]\frac{1}{2}[/tex]π + 2π

      x = [tex]\frac{1}{2}[/tex]π + [tex]\frac{4}{2}[/tex]π

      x = [tex]\frac{5}{2}[/tex]π (Tidak Memenuhi karena nilai [tex]\frac{5}{2}[/tex]π di luar interval 0 ≤ x ≤ 2π)

2. x  [tex]-\frac{1}{3}[/tex]π = [tex]-\frac{1}{6}[/tex]π + 2πk

   x = [tex]-\frac{1}{6}[/tex]π + [tex]\frac{1}{3}[/tex]π + 2πk

   x = [tex]\frac{1}{6}[/tex]π + 2πk

  Untuk nilai k = -1, maka :

      x = [tex]\frac{1}{6}[/tex]π + 2π(-1)

      x = [tex]\frac{1}{6}[/tex]π - 2π

      x = [tex]\frac{1}{6}[/tex]π - [tex]\frac{12}{6}[/tex]π

      x = [tex]-\frac{11}{6}[/tex]π (Tidak Memenuhi karena nilai [tex]-\frac{11}{6}[/tex]π di luar interval 0 ≤ x ≤ 2π)

  Untuk nilai k = 0, maka :

      x = [tex]\frac{1}{6}[/tex]π + 2π(0)

      x = [tex]\frac{1}{6}[/tex]π + 0

      x = [tex]\frac{1}{6}[/tex]π (Memenuhi karena nilai [tex]\frac{1}{6}[/tex]π di dalam interval 0 ≤ x ≤ 2π)

  Untuk nilai k = 1, maka :

      x = [tex]\frac{1}{6}[/tex]π + 2π(1)

      x = [tex]\frac{1}{6}[/tex]π + 2π

      x = [tex]\frac{1}{6}[/tex]π + [tex]\frac{12}{6}[/tex]π

      x = [tex]\frac{13}{6}[/tex]π (Tidak Memenuhi karena nilai [tex]\frac{13}{6}[/tex]π di luar interval 0 ≤ x ≤ 2π)

Maka HP : { [tex]\frac{1}{2}[/tex]π, [tex]\frac{1}{6}[/tex]π }
