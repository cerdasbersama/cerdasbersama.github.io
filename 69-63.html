4. diketahui barisan 75, 69, 63, ... tentukan Suku ke 35 tolong di bantu!​


Penjelasan dengan langkah-langkah:

Un = a + (n-1) b

= 75 + (35-1) × (-6)

= 75 + 34 × (-6)

= 75 + (-204)

= -129

Diketahui :

barisan 75, 69, 63, ...  

∴ Suku pertama (a) = 75

∴ Beda (b) = 69 - 75 = - 6

Ditanya :

Suku ke-35 atau U35

Dijawab :

Un = a + (n - 1)b

U35 = 75 + (35 - 1)(-6)

U35 = 75 + (34 × (-6))

U35 = 75 + (-204)

U35 = 75 - 204

U35 = - 129

Jadi, Suku ke-35 dari barisan tersebut adalah - 129.
