5. Lim h->0 ((x+h)^2 + (x+h) + 1 - (x^2 + x +1))/h = . . . .


[tex]\text{Hasil dari} \: \: \lim \limits_{h \to 0} \frac{ {(x + h)}^{2} + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \: \: \text{adalah} \: \: \boxed{2x + 1} \\[/tex]

Pembahasan

[tex]\text{Turunan fungsi f(x) dinotasikan dengan} \: \: f'(x) \: \: \text{atau} \: \: \frac{df(x)}{dx} \\ [/tex]

Definisi Turunan

[tex]\lim \limits_{h \to 0} \frac{f(x + h) - f(x)}{h} \\ [/tex]

Diketahui :

[tex]\text{ Limit suatu fungsi aljabar } : \: \lim \limits_{h \to 0} \frac{ {(x + h)}^{2} + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \\ \\ [/tex]

Ditanya :

[tex]\text{Hasil dari } \: \: \lim \limits_{h \to 0} \frac{ {(x + h)}^{2} + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \\ [/tex]

Jawab :

[tex] \: \: \: \: \: \lim \limits_{h \to 0} \frac{ {(x + h)}^{2} + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \\ \\ = \lim \limits_{h \to 0} \frac{ ({x}^{2} + 2xh + {h}^{2}) + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \\ \\ = \lim \limits_{h \to 0} \frac{ ({h}^{2} + 2xh + h) + ( {x}^{2} + x + 1) - ( {x}^{2} + x + 1)}{h} \\ \\ = \lim \limits_{h \to 0} \frac{ ({h}^{2} + 2xh + h) }{h} \\ \\ = \lim \limits_{h \to 0} \: (h + 2x + 1) \\ \\ = 0 + 2x + 1 \\ \\ = \boxed{2x + 1} \\ \\ [/tex]

Kesimpulan :

[tex]\text{Hasil dari} \: \: \lim \limits_{h \to 0} \frac{ {(x + h)}^{2} + (x + h) + 1 - ( {x}^{2} + x + 1)}{h} \: \: \text{adalah} \: \: \boxed{2x + 1} \\ \\ [/tex]

12. diketahui R(x) = x² + 2hx +h² , maka R(x+h) - R(x) / h


Jawab:

R(x + h) - R(x) / h = 2x + 3h

Penjelasan dengan langkah-langkah:

Diketahui

R(x) = x² + 2hx + h²

Maka,

R(x + h) = (x + h)² + 2h(x + h) + h²

= x² + 2hx + h² + 2hx + 2h² + h²

= x² + 4hx + 4h²

Setelah menentukan nilai R(x + h), segera kurangkan dari R(x).

R(x + h) - R(x) = x² + 4hx + 4h² - (x² + 2hx + h²)

= 2hx + 3h²

Setelah menentukan nilai R(x + h) - R(x), kita dengan mudah menyelesaikan R(x + h) - R(x) / h sebagaj berikut.

R(x + h) - R(x) / h = 2hx + 3h² / h

= h(2x + 3h) / h

= 2x + 3h
