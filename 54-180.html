PENDAHULUAN :

◎ Apa itu pecahan?

Pecahan adalah bagian dari keseluruhan. Pecahan terdiri atas pembilang dan penyebut. Pembilang merupakan bilangan yang dibagi, sedangkan penyebut merupakan bilangan yang membagi. Suatu pecahan dibaca dengan menyebutkan pembilang terlebih dahulu, kemudian diikuti dengan penyebut.

[tex] \boxed{\sf{ \frac{a}{b} = \frac{a \: → \: pembilang}{b \: → \: penyebut \: }}} \\ [/tex]

Macam² pecahan :Pecahan Biasa :

Terdiri dari pembilang dan penyebut. Pembilang merupakan bagian atas, dan penyebut bagian bawah. Contoh :

[tex] \boxed{\sf{ \frac{1}{2} \: ; \: \frac{2}{4} \: ; \: \frac{3}{2} \: ; \: \frac{5}{7} }}[/tex]

Pecahan Campuran :

Terdiri dari satu bilangan bulat dan satu pecahan biasa. Pecahan campuran bisa dikatakan pecahan yang sederhana dari pecahan biasa. Contoh :

[tex]\boxed{\sf{2 \frac{1}{2} \: ; \: 3 \frac{2}{8} \: ; \: 6 \frac{7}{9} \: ; \: 5\frac{2}{3} }}[/tex]

Pecahan Desimal :

Adalah pecahan yang dinyatakan dengan tanda koma. Contoh :

[tex] \boxed{\sf{0,5 \: ; \: 1,2 \: ; \: 2,3 \: ; \: 3,7}}[/tex]

Pecahan Persen :

Adalah pecahan dengan penyebut seratus dengan demikian persen berarti perseratus atau ditulis dengan lambang " % ". Contoh :

[tex] \boxed{\sf{2\% \:; \: 20\% \: ; \: 50\% \: ; \: 100\% }}[/tex]

Pecahan Permil :

Adalah pecahan dengan penyebut seribu dengan demikian Permil berarti perseribu atau ditulis dengan lambang " ‰ ". Contoh :

[tex] \boxed{\sf{300‰ \: ; \: 500‰ \: ; \: 60‰ \: ; \: 1.000‰}}[/tex]

Operasi Hitung Pecahan :

1. Penjumlahan

Caranya dengan kita mengubah penyebut kedua pecahan atau lebih menjadi sama semua. Atau bisa dilakukan dengan cara mencari KPK dari kedua bilangan atau lebih. Jika penyebut sudah sama semua, maka langsung dijumlah saja. Contoh :

[tex] \boxed{ \sf{ \frac{1}{2} + \frac{3}{4} = \frac{(1 \times 4) + (2 \times 3)}{(2 \times 4)} = \frac{10}{8} }} [/tex]

atau :

[tex]{ \sf{ \frac{2}{3} + \frac{3}{6} = \frac{4}{6} + \frac{3}{6} = \frac{7}{6} }} [/tex]

2. Pengurangan

Sama halnya dengan penjumlahan, pengurangan diharuskan mengubah penyebut kedua pecahan atau lebih menjadi sama semua. HANYA saja ini adalah operasi pengurangan dimana pengurangan adalah kebalikan dari penjumlahan. Bisa dilakukan dengan cara mencari KPK dari kedua bilangan atau lebih. Jika penyebut sudah sama semua, maka tinggal dikurangkan saja. Contoh :

[tex] \boxed{ \sf{ \frac{6}{5} - \frac{2}{3} = \frac{(6 \times 3) - (2\times 5)}{(5 \times 3)} = \frac{8}{15} }} [/tex]

atau :

[tex] \boxed{ \sf{ \frac{2}{5} - \frac{2}{10} = \frac{4}{10} - \frac{2}{10} = \frac{2}{10} }} [/tex]

3. Perkalian

Berbeda dengan pengurangan dan penjumlahan, perkalian tinggal mengalikan pembilang dengan pembilang dan penyebut dengan penyebut. Contoh :

[tex] \boxed{ \sf{ \frac{1}{4} \times \frac{1}{2} = \frac{1 \times 1}{ 4 \times 2} = \frac{1}{8} }} [/tex]

4. Pembagian

Jika pembagian pecahan, kita harus mengubah pecahan kedua yang kita bagi menjadi terbalik ( pembilang menjadi penyebut/sebaliknya ). Dan kita ubah menjadi perkalian pecahan. Contoh :

[tex] \boxed{ \sf{ \frac{3}{2} \div \frac{1}{2} = \frac{3}{2} \times \frac{2}{1} = \frac{6}{2} }} [/tex]

5. Menyederhanakan pecahan

Menyederhanakan pecahan maupun menentukan pecahan senilai dengan cara membagi pembilang dan penyebut dengan angka yang sama. Untuk pecahan campuran, kita tinggal membagi pembilang dengan penyebut. Syaratnya harus pembilang lebih besar daripada penyebut. Contoh :

Menyederhanakan :

[tex] \boxed{ \sf{ 1.) \: \frac{2}{10} = \frac{2 \div 2}{10 \div 2} = \frac{1}{5} }} [/tex]

Menentukan pecahan senilai

[tex] \boxed{ \sf{2.) \: \frac{1}{5} = \frac{1 \times 2}{5 \times 2} = \frac{2}{10 } }} [/tex]

Menentukan pecahan biasa dari pecahan campuran

[tex] \boxed{ \sf{3.) \: 3 \frac{2}{5} = \frac{5 \times 3 + 2}{5} = \frac{17}{5} }}[/tex]

Menentu pecahan campuran dari pecahan biasa

[tex] \boxed{ \sf{4.) \: \frac{17}{5} = 17 \div 5 = 3 \frac{2}{5} }} [/tex]

34. Himpunan penyelesaian dari trigonometri bentuk cos x=cos45°,untuk 0 ≤ x ≤ 360° adalaha.(50°,60°,180°) d.(50°,180°)b.(54°,180°,306°) e.(60°,180°)c.(54°,306°)Tolong bantu jawab :')​


Jawaban:

(c) (54°, 306°)

Penjelasan dengan langkah-langkah:

Kita mulai dengan mencari solusi dari persamaan trigonometri cos(x) = cos(45°). Kita tahu bahwa cos(x) sama dengan cos(y) jika dan hanya jika y adalah bilangan bulat ganjil atau genap dan x = ±y + k × 360°, di mana k adalah bilangan bulat. Dalam kasus ini, kita memiliki cos(x) = cos(45°), sehingga:

x = ±45° + k × 360° atau x = 360° - 45° + k × 360°

Dengan cara yang sama, kita dapat menentukan seluruh nilai x yang memenuhi batasan 0 ≤ x ≤ 360°. Oleh karena itu, kita perlu menyelesaikan sistem pertidaksamaan berikut:

-45° + k × 360° ≤ x ≤ 45° + k × 360° atau 315° + k × 360° ≤ x ≤ 405° + k × 360°

Dalam bentuk interval, kita dapat menuliskan solusi dari persamaan tersebut sebagai berikut:

[54°, 306°] atau [315°, 405°]

Kita perlu memeriksa setiap interval ini dan menentukan nilai-nilai x yang berada di dalam kisaran 0 ≤ x ≤ 360°. Oleh karena itu, solusi akhirnya adalah:

x = 54°, 306°, atau 360° - 45° = 315°

Jadi, jawaban yang benar adalah (c) (54°, 306°).
