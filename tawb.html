1. Untuk segitiga abc yang tidak siku-siku, buktikan bahwa tanA + tabB + tanC= tanA. tanB. tan. C.


kita tahu kalau jumlah sudut dalam segitiga adalah 180°, maka:

A + B + C = 180

dari sini bisa kita pindah ruas sudut C sehingga:

A + B = 180 - C

kita buat kedua sisi menjadi bentuk tangen

tan(A + B) = tan(180 - C)
tan(A + B) = -tan C

tan C bernilai minus karena terletak di kuadran 2. dari sini kita bisa pakai rumus tan(A + B):

[tex] \frac{tanA+tanB}{1+tanA.tanB} = -tanC [/tex]

dikalikan silang, maka jadilah:

tanA + tanB = -tanC + tanA.tanB.tanC

tanA + tanB + tanC = tanA.tanB.tanC

terbukti :) cmiiw

dalam segitiga, A + B + C = 180°

misal kita pilih, B + C = 180° - A

Gunakan rumus tan (a + b) = [tan a + tan b] / [1 - (tan a x tan b)], dan
tan (a - b) = [tan a - tan b] / [1 + (tan a x tan b)]

tan (B + C) = tan (180° - A)

[tan B + tan C] / [1 - (tan B.tan C)] = [tan 180° - tan A] / [1 + (tan 180°.tan A)]

[tan B + tan C] / [1 - (tan B.tan C)] =  [0 - tan A] / [1 + (0.tan A)]

[tan B + tan C] / [1 - (tan B.tan C)] =  - tan A

tan B + tan C = - tan A.[1 - (tan B.tan C)]

tan B + tan C = - tan A + tan A. tan B. tan C

tan A + tan B + tan C =  tan A. tan B. tan C

terbukti

21. Tuliskan dan jelaskan fungsi yang terdapat pada tab home, tab page layout, tab references, tab mailings, tab review, tab view.​


Penjelasan:

1. Home

Tab ini memiliki fungsi untuk menyunting properti-properti tulisan seperti halnya mengganti font yang digunakan dalam lembar dokumen Microsoft Word 2007, ukuran font yang diinginkan atau bahkan hingga penempatan paragraf sehingga tulisan akan menjadi lebih rapi.

2. Insert

Tab Insert ini digunakan untuk mempercantik tampilan dokumen, untuk memasukkan gambar/foto ke dalam lembar kerja, Tab Insert ini bisa dibilang tab yang mendukung Tab Home sehingga lembar kerja yang kita buat menjadi lebih menarik dan rapi.

3. Page layout

Page Layout berbeda dengan Insert, Insert mengacu pada isi dokumen, sedangkan Page Layout mengacu pada lembar itu sendiri sehingga bisa dibuat semenarik mungkin agar lembar tidak terkesan polos.

4. Referances

Tab Referensi ini berfungsi sebagai tambahan tulisan/catatan penting pada sebuah lembar kerja, misal untuk menambahkan Daftar Isi, Catatan Penting di setiap lembar kerja, dan masih banyak lagi fungsi yang masih bisa digunakan.

5. Mailings

Tab Mailings ini dapat digunakn sebagai Mail Merger dan kebutuhan surat-menyurat lainnya sehingga akan lebih efisien jika menggunakan Mailings untuk membuat surat.

6. Review

Tab review ini sangat berguna untuk mereviewkan kembali seperti melihat ulang untuk mencari kesalahan penggunaan tata bahasa, memberikan informasi mengenai jumlah kata atau bahkan bisa lagi untuk memproteksi dokumen Anda.

7. View

Tab ini memberikan bantuan tampilan pada lembar kerja Microsoft Office Word, seperti perbesaran lembar kerja, memberikan tampilan buku, dan masih banyak lagi yang bisa digunakan untuk mencoba dokumen kita dalam berbagai bentuk tampilan.

29. Semua lawak membuat tawa. Sebagian tontonan adalah lawak.A. Semua tontonan adalah lawak.B. Semua tontonan membuat tawa.C. Sebagian tontonan membuat tawa.D. Semua yang membuat tawa adalah lawak.E. Semua yang membuat tawa adalah tontonan.​


Semua lawak membuat tawa. Sebagian tontonan adalah lawak.

Kesimpulannya: Sebagian tontonan membuat tawa.

Jawaban C.

Pembahasan

Menarik kesimpulan dalam logika matematika ada 3 macam:

Modus ponens ditandai dengan adanya pernyataan majemuk implikasi dan pernyataan tunggal.Modus tollens ditandai dengan adanya pernyataan majemuk implikasi dan ingkaran dari pernyataan tunggal.Modus silogisme ditandai dengan adanya dua pernyataan majemuk implikasi.

Modus ponens

Premis 1    p ⇒ q

Premis 2    p

Konklusi     q

Modus tollens

Premis 1    p ⇒ q

Premis 2   ~q

Konklusi    ~p

Modus silogisme

Premis 1    p ⇒ q

Premis 2   q ⇒ r

Konklusi    p ⇒ r

Kata manakah yang tak sesuai dengan yang lain? dapat disimak di https://brainly.co.id/tugas/25916460

Pembahasan Soal

Semua lawak membuat tawa. Sebagian tontonan adalah lawak.

Untuk menyelesaikan soal ini kita gunakan Modus silogisme

Sebagian tontonan adalah lawak.

Semua lawak membuat tawa.

p = sebagian tontonan

q = lawak

r = tawa

Premis 1    p ⇒ q

Sebagian tontonan adalah lawak.

Premis 2   q ⇒ r

Semua lawak membuat tawa.

Konklusi    p ⇒ r

Klonklusi Sebagian tontonan membuat tawa.

37. sebutkan dan jelaskan tab tab yang ada pada tab standard!
