31. it rains all year round in indonesia ,kalimat negativenya


MAPEL : BAHASA INGGRIS
MATERI : SIMPLE PRESENT
KELAS : 7
KODE SOAL : 7.5
KODE KATEGORISASI : 7.5.5

PEMBAHASAN
================
Bentuk negative dari "it rains all year round in indonesia" adalah..

Dilihat dari struktur kalimat nya dimana setelah subject "it" menggunakan V1 dengan ditambahkan "s" yaitu RAINS, maka kalimat di atas termasuk SIMPLE PRESENT POSITIVE SENTENCE. Jadi rumus negative dari simple present adalah :
Subject + Do/Does + Not + V1 + object.

So, the answer is
=> It (DOES NOT RAIN) all year round in indonesia.

Alasan :
Menggunakan auxiliary verb DOES karena disini subject nya adalah "It". (+) it rains all year round in indonesia
(-) it *does not rain* all year round in Indonesia

#simple present form
(+) subject + verb1(-s/-es) + object
(-) subject + do/does + not + verb1 + object
(?) do/does + subject + verb 1 + object?

Semoga membantu

33. it never rains but it purs pribahasnya apa​


Mungkin purs yang kamu maksud adalah Pours.

Soal : Peribahasa dari "It never Rains But It Pours.

Jawaban : It Never Rains But It Pours maksudnya adalah jika satu masalah terjadi, maka akan diikuti masalah lainnya yang akan membuat situasi menjadi makin buruk.

Alasannya => It Never Rains itu menunjukkan suatu masalah yang dihadapi,jadi jika kita membiarkan masalah tersebut tetap turun seperti hujan,maka akan muncul(It pours) masalah lainnya yang akan membuat situasi sebelumnya menjadi makin buruk.

Mapel : Bahasa Inggris

Kelas : 7

Kode Kelas : 5

Kode Kategorisasi : 7.5
