18. Will : Sebuah ungkapan menyatakan suatu kemauan dari diri sendiri untuk melakulan sesuatu. (+) SUBJECT + WILL + VERB (Kata Kerja) + OBJECT Example : I WILL SEND A LETTER (-) SUBJECT + WILL + NOT + VERB (Kata Kerja + OBJECT Example : I WILL NOT SEND A LETTER (?) WILL + SUBJECT + VERB (Kata Kerja) + OBJECT Example : WILL YOU SEND A LETTER ? YES, I WILL NO, I WILL NOT​


Jawaban:

Will : Sebuah ungkapan menyatakan suatu kemauan dari diri sendiri untuk melakulan sesuatu.

(+) S + WILL + V1 + O

Example : I WILL SEND A LETTER

(-) S + WILL + NOT + V1 + O

Example : I WILL NOT SEND A LETTER (?) WILL + S + V1 + O

Example :

WILL YOU SEND A LETTER?

YES, I WILL / NO, I WILL NOT

Note:

S = subject (I, you, they, we, she, he, it)

V1 = kata kerja bentum pertama

O = object

NB : niatnya pengen numpang ambil poin tpi ga enak ya udah aku rapiin saja
