3. apa arti dari try again


In Indonesian language, the phrase 'try again' is best translated into 'coba lagi'. This is one of the most common and easiest phrase in English which can be translated without considering various elements.

Penjelasan:

The phrase 'try again' often pops up in a draw for example. It is one of the easiest phrase in English that can simply be translated into Indonesian literally without changing the meaning. The words included in it

try = cobaagain = lagi

So, you don't even need to find out the proper vocabulary or the context for this phrase. It is not going to be confusing for both native speaker of English and Indonesia with such translation.

13. Try a make a sentence expressing that we can do something Please answer


Soal :

Try a make a sentence expressing that we can do something ..

Pembahasan :

To state that we can do something is an important way to tell people about our skill or ability. We can make a sentence showing an ability using the following pattern.

· (+) Subject + modal verb (can) + base verb

· (-) Subject + can’t + base verb

· (?) Can + subject + base verb?

To state that we will do something, we usually use a sentence to show our plan or future activity. We can make a sentence showing what we will do by using the following pattern.

 · (+) Subject + will + base verb

 · (-) Subject + will not/won’t + base verb

 · (?) Will + subject + base verb?
