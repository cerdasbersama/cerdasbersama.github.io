32. Conditional sentences If you (drink).....coffe too much you You (stay)....awake all night


Jawaban:

type 1: If you drink coffee too much, you will stay awake all night

type 2: If you drank coffee too much, you would stay awake all night

type 3: If you had drunk coffee too much, you would have stay awake all night

Penjelasan:

Conditional Sentence Type 1

→ It is possible and also very likely that the condition will be fulfilled.

Form: if + Simple Present, will-Future

Conditional Sentence Type 2

→ It is possible but very unlikely, that the condition will be fulfilled.

Form: if + Simple Past, Conditional I (= would + Infinitive)

Conditional Sentence Type 3

→ It is impossible that the condition will be fulfilled because it refers to the past.

Form: if + Past Perfect, Conditional II (= would + have + Past Participle)
